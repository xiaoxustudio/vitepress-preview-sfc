# i18n / 国际化

ViewSfc 组件内置 `en` 和 `zh` 两种语言的国际化支持。

## 自动检测

语言会根据 `<html lang>` 属性自动检测。在 VitePress 中，此属性由 [locales 配置](https://vitepress.dev/reference/site-config#locales) 设置。

```
// <html lang="zh-CN"> → 使用中文
// <html lang="en"> → 使用英文
// 未设置 → 英文（默认）
```

## 手动切换语言

```ts
import {
	ViewSfcConfigFn,
	ViewSfcTagSymbol
} from "@vitepress-preview-sfc/components";
import { provide } from "vue";

const config = ViewSfcConfigFn();
provide(ViewSfcTagSymbol, config);

// 切换到中文
config.setLocale("zh");

// 切换回英文
config.setLocale("en");
```

## 注册自定义语言

```ts
const config = ViewSfcConfigFn();

config.locales.jp = {
	collapseText: "折りたたむ",
	copyTextSuccess: "コピー成功",
	copyTextError: "コピー失敗",
	accessibility: {
		btnGroupLabel: "コードプレビューコントロール",
		codeRegionLabel: "ソースコードビューア",
		collapseBtnLabel: "コードを折りたたむ"
	}
};

config.setLocale("jp");
```

## API

| 属性                | 类型                             | 说明               |
| ------------------- | -------------------------------- | ------------------ |
| `locale`            | `Ref<string>`                    | 当前语言名称       |
| `locales`           | `Record<string, LocaleMessages>` | 已注册的语言包     |
| `setLocale(locale)` | `(name: string) => void`         | 切换到已注册的语言 |
| `collapseText`      | `Ref<string>`                    | 折叠按钮文字       |
| `copyTextSuccess`   | `Ref<string>`                    | 复制成功提示       |
| `copyTextError`     | `Ref<string>`                    | 复制失败提示       |
